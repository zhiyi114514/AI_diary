# 🎭 AI日记

一款由AI提供情感陪伴的个人日记应用，每天记录生活，让AI化身亲密好友，给予温暖回复。

## ✨ 功能特性

- **✍️ 写日记** - 每天记录生活和心情，AI会给予温暖、贴心的回复
- **📋 待办清单** - 管理日常任务，已完成的项目会自动保留3天后删除
- **😊 心情日历** - 可视化查看每天的情绪变化，支持月份切换浏览
- **📊 自动总结** - 周报（周一自动生成）和月报（每月1号自动生成），记录情绪轨迹
- **📦 导入导出** - 支持多种导出格式，随时备份你的日记数据
- **🧠 人物知识库** - AI会自动记住你日记中的重要人物及其信息，提供更贴心的回复

## 🚀 快速开始

## 网站
https://ai-diary.streamlit.app

### 首次使用

1. 打开应用后，进入 **设置 > AI设置**
2. 配置 LLM API：
   - **API Base URL** - 例如 `https://api.openai.com`（或其他兼容OpenAI格式的服务）
   - **API Key** - 你的API密钥
3. 点击 **获取可用模型** 验证配置
4. 配置 **天气设置**（可选）：
   - 从中国天气网获取你所在城市的URL
   - 支持实时天气信息显示
5. 开始写日记 ✍️

### 配置说明

#### AI设置

| 项目 | 说明 |
|------|------|
| API Base URL | OpenAI、Claude、Ollama等兼容OpenAI格式的LLM服务地址 |
| API Key | 对应服务的API密钥 |
| 回复随机性 | 调整AI回复的创意度（0.0-2.0） |
| 历史日记数量 | 每次发送给AI的历史日记篇数，越多越贵但AI理解更深 |
| AI人设 | 自定义AI的角色设定，修改后回复风格会随之变化 |
| 用户自我介绍 | 告诉AI关于你的信息，让回复更贴心 |

#### 总结设置

- **周报** - 周一自动生成（需≥5篇日记）
- **月报** - 每月1号自动生成（需≥25篇日记）
- 支持编辑和保存修改后的总结内容

#### 知识库设置

- AI会自动记录日记中新出现的重要人物
- 知识库通过正则检索触发
- 在 **设置 > 知识库** 中查看和管理所有人物信息
- 支持手动添加、编辑或删除人物记录
- 若有其他想让AI了解的信息（如游戏机制），也可以手动添加到知识库

## 🔒 隐私与安全说明

### 数据存储

✅ **本地存储**

- 所有数据（日记、待办、心情、配置等）存储在你的浏览器本地（localStorage）
- **完全不上传到任何服务器**
- 不同用户/浏览器的数据完全隔离

✅ **API密钥保护**

- API Key 以 Base64 编码方式存储在浏览器本地
- 仅用于向你配置的LLM服务发起API请求
- 不会被记录或发送到第三方

⚠️ **使用提醒**

**请勿在公共电脑上使用本应用！**

公共电脑（网吧、学校、公司等共用设备）存在以下风险：

- 其他用户可访问浏览器数据和localStorage
- 系统管理员可能有权限查看本地存储内容
- 恶意软件可能窃取存储的API密钥
- 你的日记（隐私内容）可能被他人查看

**建议在以下设备上使用：**

- 个人电脑（仅你使用）
- 个人手机
- 其他只有你有访问权限的设备

### 数据备份

- 定期在 **导入导出 > 导出** 中下载完整数据备份（JSON格式）
- 备份不包含 API Key，安全性有保障
- 妥善保管备份文件
